# Íntegra Consultoria

Landing page v1 para captação de leads da Íntegra Consultoria.

## Rodar localmente

```bash
npm install
npm run dev
```

Configure o número do WhatsApp em `.env.local`:

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=5511999999999
```

## Escopo do v1

- Landing page em Next.js.
- CTA direto para WhatsApp.
- Formulário de triagem que monta uma mensagem estruturada para WhatsApp.
- Copy com fronteira regulatória conservadora: diagnóstico, educação técnica e suporte operacional, sem recomendação personalizada de novos investimentos.
